# Intership-project
PROJECT TITLE:
Digital evidence management and Crime Record Tracking System 

Digital Evidence Management and Crime Record Tracking System is a web-based application designed to securely store, manage, and track crime records and digital evidence. The system helps police or investigation departments organize case details, upload evidence files, monitor case progress, and improve data management efficiently.


Project Overview:

This project is a Python-based console application designed to help police departments manage crime records and digital evidence efficiently. The system allows officers or administrators to store, update, search, and track criminal case information in an organized way.

The application focuses on managing investigation details such as case IDs, suspect information, evidence records, assigned officers, and case status. It also provides search and filtering features to quickly find cases and generate investigation reports.

The system stores data permanently using JSON or CSV files so records remain available even after the program is closed.

Main Features:

Add and manage crime cases

1)Track digital evidence 
2)Update investigation status
Search and filter crime records
3)Generate investigation reports
4)Save and load records automatically
5)Handle invalid or duplicate data safely
